Core Topics

1. Introduction to Data Communication

Data communication is the process of sending information between a sender and a receiver using a communication channel (medium) and a set of rules (protocol).

2. Networking Fundamentals

Networking fundamentals explain how devices communicate across a network.

3. Network & Network Reference Models

Reference models like the OSI model help break communication into layers.

4. Transmission Media

Transmission media are the physical paths that carry signals.

5. Signals

Signals describe how information is represented over time.

6. Wireless Networks and Technologies

Wireless networking covers communication without physical cables.

7. CIA Triad and Cryptography

The CIA triad—Confidentiality, Integrity, and Availability is a foundation for security goals.

8. Data Security, Network Security and Cybersecurity

Cybersecurity focuses on defending systems and networks against threats.
